Surgimento e história da internet

Surgimento da internet

Voltar

A Internet surgiu no ano de 1969 criada com objectivo de possibilitar comunicação nos tempos de guerra Após o período de guerra. Arpanet uma agencia que passou a ser mais pesquisada e desenvolvida e desde então, tornou-se um meio de comunicação apenas dentro do universo acadêmico por mais de vinte anos. A partir da década de 90, foi criado o WWW que a gente conhece. O World Wibe Web foi desenvolvido por um engenheiro inglês chamado Tim Bernes-Lee. Depois disso, foi criado o HTTP que permite o envio de dados codificados. Desde então, surgiram os sites cada vez mais atraentes e com interfaces gráficas cada vez mais desenvolvidas.

Criador

Voltar

Timothy John Berners-Lee nasceu em Londres, Inglaterra, em 8 de junho de 1955. Ele é um físico britânico, cientista da computação. Seu cargo mais conhecido é ser o criador da World Wide Web, o famoso termo técnico WWW ou seja a Internet.

O que é o HTML

Voltar

O HTML é uma linguagem de marcação com a qual os elementos de uma página são identificados e definidos, geralmente, por um par de marcadores conhecidos como tags. Esses elementos podem ser desde textos e suas formatações, como parágrafos, sublinhados e links, até conteúdos multimídia, como imagens e vídeos.

Função do HTML

Voltar

O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web

Tags

Tags são elementos que nos permitem orientar o computador, os seja é uma forma de dar instruções para o computador realizar na tela de computadores
